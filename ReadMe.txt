This is a guide of files related to 'Unihome'-a platform present with Colleges/Universities database related to IT Courses.

1)'public' : Folder contains all front-end related files (html,css and script files)
2)'node_modules' : Folder contains modules which required for creating back-end
3)'app.js' : File contains Back-end code where everything is integrated
4)'database.sql' : File contains database in SQL
5)'database.js' : Used to export the database for connection
6)'Unihome.png' : Screenshot of the Client-Side Page
